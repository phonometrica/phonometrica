Annotations¶

This page documents the Annotation type.

General concepts¶

An Annotation object represents a time-aligned annotation of a sound file, on one or more layers.

Functions¶

Annotation.new(pattern)¶

Create and return a new regular expression (regex) from a string pattern. The regex can be matched against any string.

local re = regex.new("^(..)")
-- Do something with re...

See also: pattern()
