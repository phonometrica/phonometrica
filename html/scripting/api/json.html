JSON¶

General concepts¶

Phonometrica objects can be converted to JSON (Javascript Object Notation), a lightweight declarative format which is often used to store data. See http://www.json.org.

Functions¶

json.stringify(object)¶

Convert object to a string according to the JSON specification.

var o = {
    "name": "John",
    "pi": 3.14
}

var s = json.stringify(o)
print(s)
json.parse(str)¶

Convert the string str to an object according to the JSON specification.

var s = '{"name":"John", "pi":3.14}'
var o = json.parse(s)
print("The value of pi is " + o.pi)
