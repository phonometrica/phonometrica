Classes and instances¶

Overview¶

Phonometrica’s scripting language is an object-oriented programming language. This means that all values, including primitive values such as integers and Booleans, are objects on which we can perform computations using functions. Every type in Phonometrica is represented by a class, which is a “blueprint” that allows us to create instances of the type it defines:
